# bash-cli
Python script for reading bash.org.pl quotes. 

##Usage

 `./bash-cli.py quote_id` 
 
 To view a specified quote
 
 
 `./bash-cli.py -r`
 
 To randomly select a quote
 
 ##Requirements 
 
 BeautifulSoup4
