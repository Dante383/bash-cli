# bash-cli
Python script for reading bash.org.pl quotes. 
Usage: 
 ./bash-cli.py quote_id  
 To view a quote from the ID
    Or:
 ./bash-cli.py -r 
 To randomly select a quote
 

Made on linux, require bs4 package to work. Can not work on windows, but not checked
